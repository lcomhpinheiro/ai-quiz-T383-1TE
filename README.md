# 🧠 AI Quiz Generator

A simple AI-powered quiz generation and solving system.  
The user provides a topic, the AI generates questions automatically, and the answers are then submitted for evaluation via a backend service.
It can be used by students to help them understand topics they don't yet understand very well

---

## 🚀 Features

- AI-generated quizzes based on a topic
- Support for multiple-choice and open-ended questions
- Dynamic quiz rendering on the frontend
- Answer submission to backend for evaluation
- JSON response with results

---

## 🖥️ Demo Flow

1. Enter a topic in the input field
2. Click **"Generate"**
3. Answer the generated questions
4. Click **"Submit answers"**
5. View the evaluation result returned by the backend

---
## 🧩 Technologies Used
- HTML5
- Vanilla JavaScript
- Fetch API
- External Node.js + AI backend
